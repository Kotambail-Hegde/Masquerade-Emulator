daid/​ppu_scanline_bgp.gb (DMG)Mid scanline BGP register changes. Requires accurate PPU timing. Changing the BGP register has three possible effects for one pixel: the previous BGP is used, the next BGP is used, or the OR result of the previous and next BGP is used, the last case causing a black line. Which case occurs seems to be hardware and instance dependent as some DMGs do not do a consistent single case. PASS
daid/​ppu_scanline_bgp.gb (GBC)Mid scanline BGP register changes. Requires accurate PPU timing. Changing the BGP register will have the next BGP colors used. FAIL
